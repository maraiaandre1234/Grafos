# Mi Blog sobre Grafos

Este es un blog técnico dedicado a la estructura de datos Grafos. Aquí encontrarás artículos que cubren los aspectos fundamentales de los grafos, incluyendo su definición, representación y algoritmos de recorrido.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **index.html**: Página principal del blog que contiene enlaces a los artículos y navegación.
- **about.html**: Información sobre el blog, su propósito y el autor.
- **posts/**: Carpeta que contiene los artículos del blog.
  - **01-introduccion-grafos.html**: Introducción a los grafos, incluyendo nodos, aristas y tipos.
  - **02-representacion-grafos.html**: Representación de grafos en memoria, incluyendo lista de adyacencia y matriz de adyacencia.
  - **03-recorridos-grafos.html**: Algoritmos fundamentales de recorrido, como BFS y DFS.
- **css/**: Carpeta que contiene los estilos del blog.
  - **styles.css**: Archivo de estilos que define la apariencia del blog.
- **js/**: Carpeta que contiene scripts de JavaScript opcionales.
  - **main.js**: Archivo de JavaScript para interactividad o contenido dinámico.
- **.github/**: Carpeta que contiene la configuración para GitHub Actions.
  - **workflows/**: Carpeta para flujos de trabajo de GitHub.
    - **github-pages.yml**: Configuración para desplegar el blog en GitHub Pages.
- **LICENSE**: Información sobre la licencia del proyecto.
